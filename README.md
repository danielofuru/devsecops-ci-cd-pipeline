# 🚀 DevSecOps CI/CD Pipeline  

## 📌 Overview  
This project demonstrates a complete DevSecOps pipeline integrating CI/CD, security, and cloud deployment.

---

## ⚙️ Tech Stack  
- Jenkins  
- GitHub Actions  
- Docker  
- AWS EC2  
- Terraform  
- Trivy (Security Scanning)  

---

## 🔄 Pipeline Flow  

1. Developer pushes code to GitHub  
2. GitHub Actions triggers pipeline  
3. Jenkins handles build and test stages  
4. Docker image is created  
5. Security scan runs using Trivy  
6. Application is deployed to AWS EC2  

---

## 🔐 Security Features  
- Vulnerability scanning integrated into pipeline  
- Secure deployment practices  
- Automated checks before deployment  

---

## 📊 Key Achievements  
- Reduced manual deployment effort by 80%  
- Automated full build and deployment lifecycle  
- Improved deployment consistency and reliability  

---

## 🏗️ Architecture  

GitHub → GitHub Actions → Jenkins → Docker → AWS EC2  

---

## ▶️ How to Run  
1. Clone the repo  
2. Configure Jenkins  
3. Set up AWS credentials  
4. Run pipeline
